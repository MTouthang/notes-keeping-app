## Tech stack
***
- javascript
- Backend
    - Mongoose
    - Mongodb Atlas
    - Express
    - Node
    - Bcryptjs
    - yamljs
    - validator
    - swagger-ui-express
    - nodemon
    - morgan
    - jsonwebtoken
    - dotenv
    - cors
    - cookie-parser
- FrontEnd 
    - React
    - React toastify
    - Axios
    - react-color
    - material ui
